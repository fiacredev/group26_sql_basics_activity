# ALU Database — Group 26

MySQL school database (`alu_db`) with students, classrooms, faculty, courses, and extracurricular activities.

## Files

- `alu_db.sql` — all CREATE, INSERT, UPDATE, DELETE, and SELECT statements

## Tables


| Table                               | Owner    |
| ----------------------------------- | -------- |
| Students                            | Member A |
| Classroom                           | Member B |
| Faculty                             | Member C |
| Courses                             | Member D |
| Extra_Curricular_Activities         | Member E |
| Student_Courses, Student_Activities | Member E |


## Run

```bash
mysql -u your_username -p < alu_db.sql
```
